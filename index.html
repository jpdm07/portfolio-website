UX Redesign Concept

Redesign of an existing interface to improve usability and user satisfaction.

Problem

A content-heavy web interface had poor navigation and low user satisfaction. Goal: improve clarity, task flow, and overall usability.

My Role

Conducted user research, created wireframes and prototypes, and applied design principles to enhance usability. Focused on accessibility and intuitive navigation.

Process
  • Analyzed current interface and identified user pain points.
  • Conducted competitor benchmarking and best-practice research.
  • Created wireframes and interactive prototypes in Figma.
  • Tested prototypes with sample users and iterated based on feedback.
  • Applied accessible design practices for typography, color, and layout.
Outcome

Resulted in a cleaner, more intuitive interface with improved navigation and task completion. Feedback indicated users felt the experience was simpler and more enjoyable.
